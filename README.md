# Task Management App (Flutter + Firebase+ Shared prefrences)

A simple task management app built with Flutter that uses Firebase Authentication for user accounts and SharedPreferences for storing to‑do data locally.

## Features

- Sign up with email and password  
- Email verification for new users  
- Log in with email and password  
- Reset password via email  
- Firebase Authentication integration  
- Add new tasks  
- Mark tasks as finished / unfinished  
- Delete tasks  
- Filter tasks by finished / unfinished  
- To‑do list data stored locally using SharedPreferences  
- Used FlutterToast for better understandings

## Tech Stack

- **Framework:** Flutter  
- **Language:** Dart  
- **Backend/Auth:** Firebase Authentication  
- **Local Storage:** SharedPreferences
